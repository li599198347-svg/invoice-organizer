# 差旅发票整理技能 - 技能简报

## 📋 技能概述

从 QQ 邮箱自动提取差旅发票，生成 Excel 汇总表和 Word 打印版。

**版本：** v1.2  
**最后更新：** 2026-03-29  
**测试数据：** 2026-02-01 至 2026-03-31，共 19 条记录，总金额 ¥4,247.27

---

## ✅ 支持的发票类型

| 类型 | 识别方式 | 提取内容 | 状态 |
|------|---------|---------|------|
| **滴滴** | 发件人 `xiaojukeji.com` | 发票号、金额、行程日期、起点、终点 | ✅ |
| **阳光出行** | 主题含"第三方"/"阳光出行" | 发票号、金额、行程日期、起点、终点 | ✅ |
| **通行费** | 发件人 `txffp.com` | 多张发票号、总金额、行程日期、入口站、出口站 | ✅ |
| **12306 火车票** | 主题含"火车票"/"高铁"/"12306" | 发票号、金额、行程日期、出发站、到达站 | ✅ |
| **携程机票** | 主题含"携程"或 `trip.com` | 发票号、金额（简化处理） | ✅ |

## 🔧 核心处理逻辑

### 1. 邮件扫描与类型识别

```python
# 优先级：发件人域名 > 主题关键词
if 'xiaojukeji.com' in sender:
    inv_type = '阳光出行' if '第三方' in subject else '滴滴'
elif 'txffp.com' in sender:
    inv_type = '通行费'
elif '12306' in sender or '火车票' in subject:
    inv_type = '12306 火车票'
elif '携程' in subject or 'trip.com' in sender:
    inv_type = '携程机票'
```

### 2. 行程单提取

```
支持两种格式：
1. 多行格式：城市、起点、终点各占一行
2. 单行格式：所有内容在一行，用空格分隔

提取步骤：
1. 从行程单提取行程日期（滴滴/通行费格式不同）
2. 找到"里程"数字行
3. 向前找序号
4. 序号和里程之间的内容：车型→时间→城市→起点→终点
5. 城市特征：常见城市名或含"市"的短词
```

### 3. 12306 火车票提取

```python
# 正则：车次号 1-4 位 + 年份 4 位 + 日期
pattern = r'([GDKZTC]\d{1,4})(\d{4})\s*年\s*(\d{1,2})\s*月\s*(\d{1,2})\s*日'
# 站点：从 PDF 提取所有"站"，反转顺序（到达站在前，出发站在后）
```

### 4. 通行费 ZIP 处理

```
1. 解压 ZIP，遍历所有 XML 文件
2. 提取 <EIid>发票号</EIid>
3. 累加所有子发票金额
4. 保存所有子发票 PDF
5. Word 中每张子发票单独成页
```
