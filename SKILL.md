# 差旅发票整理技能

从 QQ 邮箱自动提取发票，生成报销用 Excel 和 Word 打印版。

## 核心功能

### ✅ 已实现

| 发票类型 | 识别方式 | 附件处理 | 行程提取 | 状态 |
|---------|---------|---------|---------|------|
| **滴滴** | 发件人含 `xiaojukeji.com` | 发票 PDF + 行程单 PDF | ✅ 从行程单提取起点终点 | ✅ |

## 配置

### QQ 邮箱 IMAP 设置

1. 登录 QQ 邮箱 → 设置 → 账户 → POP3/IMAP/SMTP 服务
2. 开启 **IMAP/SMTP 服务**
3. 生成 **授权码**（不是邮箱密码）

### 写入记忆文件

在 `~/.openclaw/workspace/MEMORY.md` 中添加：

```markdown
## 邮件配置
IMAP_HOST=imap.qq.com
IMAP_PORT=993
IMAP_USER=599198347@qq.com
IMAP_PASS=你的授权码
IMAP_MAILBOX=INBOX
```

## Word 打印版格式

### 页面设置
- **纸张**：A5 横向（210mm × 148mm）
- **边距**：8mm 四边等距
- **每页**：一张发票 PDF 图片

### 页脚格式
- **位置**：右下角
- **字体**：Arial 7pt
- **内容**：`Page X - 发票号 (n/总数)`
  - 单张发票：`Page 1 - 滴滴电子发票`
  - 多张发票：`Page 1 - 26317907100300133055 (1/4)`

### 通行费处理
- **Excel**：一张汇总发票对应一行，发票号列换行列出所有子票据号
- **Word**：每张子发票单独一页，便于粘贴到报销单

## 处理逻辑详解

### 1. 邮件扫描与类型识别

```python
# 优先级：发件人域名 > 主题关键词
if 'xiaojukeji.com' in sender:
    inv_type = '阳光出行' if '第三方' in subject else '滴滴'
elif 'txffp.com' in sender:
    inv_type = '通行费'
elif '火车票' in subject or '高铁' in subject:
    inv_type = '火车票'  # 待实现
elif '机票' in subject or '航空' in subject:
    inv_type = '飞机票'  # 待实现
```

### 2. 行程单提取逻辑

```
行程单 PDF 结构：
├── 表头：序号 | 车型 | 上车时间 | 城市 | 起点 | 终点 | 里程 | 金额
└── 数据行：1 | 滴滴特快 | 03-15 14:49 | 深圳市 | 蛇口邮轮中心... | 宝安机场... | 25.3 | 69.90

提取步骤：
1. 找到"里程"数字行（如 25.3）
2. 向前找序号（如 1）
3. 序号和里程之间的内容：车型→时间→城市→起点→终点
4. 城市特征：短词含"市"（如"深圳市"）
5. 城市之后的内容平分：前半=起点，后半=终点
6. 从"行程起止日期"提取实际行程日期（优先于邮件日期）
```

### 3. 多发票↔多行程单匹配

```
邮件 51 示例（一张发票对应多个行程）：
├── 滴滴电子发票 A.pdf (¥1682.98)
├── 滴滴电子发票 B.pdf (¥90.91)
├── 滴滴出行行程报销单 A.pdf (19 个行程)
└── 滴滴出行行程报销单 B.pdf (2 个行程)

匹配规则：
- 文件名含 A → 发票 A 对应 行程单 A → Excel 显示"对应 19 个行程"
- 文件名含 B → 发票 B 对应 行程单 B → Excel 显示"对应 2 个行程"
- 无字母后缀 → 所有行程归到第一张发票
```

### 4. 通行费 ZIP 处理

```
ZIP 内容：
├── xml/26317907100300133055.xml  # 发票数据
├── xml/26317907090300037055.xml
├── pdf/26317907100300133055.pdf  # 发票 PDF
└── pdf/26317907090300037055.pdf

提取逻辑：
1. 遍历所有 XML 文件
2. 提取 <EIid>发票号</EIid>
3. 提取 <TotaltaxIncludedAmount>金额</TotaltaxIncludedAmount>
4. 累加所有子发票金额
5. 保存所有子发票 PDF 到"发票 PDF"文件夹
6. Word 中每张子发票单独一页
```

## 常见问题

### Q: 行程单提取不到路线信息
**A:** 检查行程单 PDF 格式是否标准。非标准格式会显示"详见发票"。

### Q: 通行费 Word 中只显示第一张发票
**A:** 已修复。现在通行费 ZIP 中所有子发票都会在 Word 中单独成页。

### Q: 一封邮件多张发票只处理了一张
**A:** 已修复。现在支持文件名 A/B 后缀的配对逻辑。

### Q: 如何修改输出目录
**A:** 修改 `scripts/main.py` 中的 `OUTPUT_DIR` 变量：

OUTPUT_DIR = os.path.join(os.path.expanduser('~/Desktop'), '差旅发票')
