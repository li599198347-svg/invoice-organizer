# 差旅发票整理技能 - 版本更新日志

## v1.2 (2026-03-29) - 完整功能版本 🎉

### ✅ 新增功能

#### 1. 12306 火车票支持
- 自动识别 12306 邮件（主题含"火车票"/"高铁"/"铁路"或发件人含"12306"）
- 从发票 PDF 提取：
  - 发票号码（20 位）
  - 车次号（G/D/K/Z/T/C 开头）
  - 行程日期
  - 出发站/到达站
- 正则优化：支持无空格格式（如"G83632026 年"）

#### 2. 通行费行程日期
- 从行程单 PDF 提取"通行日期"
- 格式转换：20260310 → 2026-03-10
- Excel 中正确显示行程日期列

#### 3. 携程机票简化处理
- 从文件名提取订单号
- 只提取发票号和金额
- 出发地显示"详见发票"（节省处理时间）

### 🔧 核心优化

#### 行程单提取
- 支持两种格式：
  - **多行格式**：城市、起点、终点各占一行
  - **单行格式**：所有内容在一行，用空格分隔
- 城市识别：常见城市名列表 + 含"市"的短词
- 里程行识别：跳过金额行，只处理第一个里程数字

#### 12306 处理
- 正则表达式：`([GDKZTC]\d{1,4})(\d{4})\s*年\s*(\d{1,2})\s*月\s*(\d{1,2})\s*日`
- 站点提取：从 PDF 提取所有"站"，反转顺序（到达站在前，出发站在后）
- 站点过滤：排除"开票"、"电子客票"等干扰词

#### 多发票邮件处理
- 滴滴 A/B 发票 → 行程单 A/B 配对
- 通行费 ZIP → 所有子发票 PDF 单独保存
- Word 中每张子发票单独成页

#### PDF 命名规范
- 所有发票 PDF 以发票号命名
- 行程单 PDF 以"邮件序号_原文件名"命名
- 避免文件覆盖，便于查找

### 📄 输出文件

```
~/Desktop/差旅发票/
├── 发票 PDF/              # 按发票号命名
├── 行程单 PDF/            # 按邮件序号_原文件名
├── 差旅发票汇总.xlsx      # Excel 汇总表（8 列）
└── 差旅发票汇总打印.docx  # Word 打印版（A5 横向）
```

### Excel 列说明

| 列 | 字段名 | 说明 |
|---|--------|------|
| A | 序号 | 按类型分组计数 |
| B | 类型 | 滴滴/阳光出行/通行费/携程机票/12306 火车票 |
| C | 发票号码 | 多张子票据换行显示 |
| D | 行程日期 | 从发票或行程单提取 |
| E | 出发地/出发站 | 具体地点 或 "对应 X 个行程" 或 "详见发票" |
| F | 到达地/到达站 | 具体地点 或 空 或 "详见发票" |
| G | 金额 (元) | 数字格式，2 位小数 |
| H | Word 页码 | 对应打印版页码 |
